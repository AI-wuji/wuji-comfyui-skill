## 🔧 融合核心原则

**不是复制代码，而是直接引用！**

```python
# ✅ 正确做法
import comfy.sd
from nodes import LoraLoader

# 直接调用官方API
model = comfy.sd.load_diffusion_model(unet_path)

# 直接实例化官方节点
lora_loader = LoraLoader()
model, clip = lora_loader.load_lora(...)
```

**好处：原插件更新，融合节点自动更新！**

### 工具使用原则
